¿Qué es SQL?

Introducción

SQL (Structured Query Language) es un lenguaje específico diseñado para gestionar y manipular bases de datos relacionales. Fue desarrollado por IBM en los años 70 y se ha convertido en el estándar para trabajar con sistemas de bases de datos como MySQL, PostgreSQL, SQL Server, SQLite, Oracle, entre otros.

Más Informarción

Características principales

  • Permite consultar, insertar, actualizar y eliminar datos en bases de datos.

  • Usa una sintaxis estructurada basada en comandos.

  • Trabaja con tablas relacionadas entre sí.

  • Es declarativo: le dices qué hacer, no cómo hacerlo.

Comandos

Comando Función
SELECT Recupera datos de una tabla
INSERT Agrega nuevos registros
UPDATE Modifica datos existentes
DELETE Elimina registros
CREATE Crea bases de datos, tablas u otros objeto
DROP Elimina bases de datos o tablas
WHERE Filtra registros según condiciones
JOIN Combina datos de varias tablas

Ventajas y Desventajas

Ventajas

  • Ideal para almacenar grandes volúmenes de datos.

  • Muy eficiente para realizar búsquedas, filtros y reportes.

  • Lenguaje estándar en el mundo empresarial y de software.

Desventajas

  • No está diseñado para tareas fuera del ámbito de bases de datos.

  • Requiere entender bien la estructura de las tablas y relaciones.

  • Algunas variaciones según el sistema (MySQL ≠ Oracle en algunos detalles).

Ejemplo!


-- Crear una tabla
CREATE TABLE usuarios (
id INT PRIMARY KEY,
nombre VARCHAR(50),
edad INT
);
-- Insertar datos
INSERT INTO usuarios (id, nombre, edad)
VALUES (1, 'user', 18);
-- Consultar datos
SELECT * FROM usuarios;
-- Filtrar datos
SELECT nombre FROM usuarios WHERE edad > 12;
