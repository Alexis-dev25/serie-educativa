Python · Nivel básico

Aprende a instalar Python, crear tus primeros scripts y utilizar la terminal para ejecutarlos en Windows, Linux y macOS.

Nivel: Básico Ideal para empezar Compatibilidad: Windows, Linux, macOS

1. Instalación de Python

Windows

  • Entra en la web oficial de Python y descarga la última versión.
  • Marca la casilla "Add Python to PATH" durante la instalación.
  • Abre PowerShell y ejecuta python --version para comprobarlo.

Linux (ejemplo Ubuntu)

  • Abre la terminal.
  • Ejecuta sudo apt update y luego sudo apt install python3 python3-pip.
  • Comprueba con python3 --version.

macOS

  • Instala Homebrew si no lo tienes.
  • Ejecuta brew install python.
  • Verifica con python3 --version.

2. Tu primer programa

Hola mundo

Crea un archivo llamado hola.py con este contenido:

print("Hola, mundo desde Python")

Ejecutar desde la terminal

  • Navega a la carpeta donde guardaste el archivo.
  • Ejecuta python hola.py (o python3 en Linux/macOS).
  • Deberías ver el mensaje en pantalla.
