Java

¿Qué es Java?

Introducción

Java es un lenguaje de programación orientado a objetos, de propósito general y compilado, desarrollado por Sun Microsystems (ahora Oracle) en 1995. Es famoso por "Write Once, Run Anywhere" (WORA) gracias a la JVM.

Java es uno de los lenguajes más utilizados en el mundo empresarial, especialmente en aplicaciones de gran escala, sistemas backend y desarrollo Android.

Más Información

Características principales

  • Lenguaje compilado e interpretado (bytecode en JVM).

  • Fuertemente tipado y orientado a objetos.

  • Soporta modularidad, multihilos, concurrencia y seguridad.

  • Multiplataforma: funciona en cualquier sistema con JVM.

  • Gestión automática de memoria con Garbage Collector.

  • Amplio ecosistema de librerías y frameworks.

Usos comunes

  • Aplicaciones Android y desarrollo móvil.

  • Sistemas empresariales y aplicaciones web.

  • Frameworks como Spring Boot, Jakarta EE.

  • Big Data (Hadoop, Spark), IoT, sistemas embebidos.

  • Juegos y herramientas de escritorio.

Ventajas y Desventajas

Ventajas

  • Portabilidad con WORA y JVM.

  • Excelente gestión de memoria y errores.

  • Comunidad enorme con 25+ años de madurez.

  • Altísima demanda laboral en el sector empresarial.

Desventajas

  • Sintaxis verborreica comparada con Python.

  • Mayor consumo de memoria que C++ o Go.

  • Configuración más compleja para tareas simples.

🎯 Ejemplos Complejos

Ejemplo 1: Gestor de Estudiantes

public class Estudiante {
    private String nombre;
    private double promedio;
    
    public Estudiante(String nombre, double promedio) {
        this.nombre = nombre;
        this.promedio = promedio;
    }
    
    public boolean estáAprobado() { return promedio >= 6.0; }
}

ArrayList estudiantes = new ArrayList<>();
estudiantes.add(new Estudiante("Juan", 7.5));

double promedio = estudiantes.stream()
    .mapToDouble(e -> e.promedio)
    .average()
    .orElse(0.0);

Ejemplo 2: Validador de Datos

public class Validador {
    public static boolean esEmail(String email) {
        return email.matches("^[A-Za-z0-9+_.-]+@(.+)$");
    }
    
    public static boolean esContraValid(String pass) {
        return pass.length() >= 8 &&
               pass.matches(".*[A-Z].*") &&
               pass.matches(".*\\d.*");
    }
}

System.out.println(Validador.esEmail("user@mail.com"));
System.out.println(Validador.esContraValid("Segura123"));

Ejemplo 3: Calculadora

public class Calculadora {
    private ArrayList historial = new ArrayList<>();
    
    public double sumar(double a, double b) {
        double res = a + b;
        historial.add(a + " + " + b + " = " + res);
        return res;
    }
    
    public void mostrarHistorial() {
        for (String op : historial) {
            System.out.println(op);
        }
    }
}

Calculadora calc = new Calculadora();
calc.sumar(5, 3);
calc.mostrarHistorial();
