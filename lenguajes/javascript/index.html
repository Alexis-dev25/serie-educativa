JavaScript · Nivel intermedio

JavaScript te permite crear interfaces dinámicas en la web y también aplicaciones de servidor con Node.js.

Nivel: Intermedio Front-end y back-end Compatible con todos los sistemas

1. Entorno para JavaScript

Navegador + editor

  • Utiliza un navegador moderno (Chrome, Edge, Firefox).
  • Instala un editor como VSCode para escribir el código.
  • Crea un archivo index.html y enlaza un script app.js.

Node.js (para back-end)

  • Descarga Node.js desde su web oficial.
  • Verifica la instalación con node --version y npm --version.
  • Crea un proyecto con npm init -y y ejecuta node app.js.

2. Conceptos clave

Ejemplo básico

const nombre = "Alex";
console.log(`Hola, ${nombre}`);

Qué debes dominar

  • Variables (let, const).
  • Funciones y arrow functions.
  • Arrays, objetos y métodos más comunes.
  • Manipulación del DOM y eventos.
  • Uso de librerías y paquetes con npm.

Combina esta guía con la sección de editores (VSCode) y sistemas (Windows/Linux) para tener un flujo de trabajo completo.
