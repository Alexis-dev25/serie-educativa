JavaScript

¿Qué es JavaScript?

Introducción

JavaScript (JS) es un lenguaje de programación interpretado, ligero y orientado a objetos, ampliamente usado en el desarrollo web. Fue creado por Brendan Eich en 1995 y originalmente diseñado para ejecutarse en navegadores web, aunque hoy en día también se usa en el servidor (gracias a Node.js)

Más Informarción

Características principales

  • Lenguaje de scripting del lado del cliente (aunque también del servidor con Node.js).

  • Interpretado y ejecutado directamente por los navegadores.

  • Orientado a objetos, basado en prototipos.

  • Funciona de manera asíncrona gracias a funciones como setTimeout, fetch y el uso de promesas y async/await.

  • Es uno de los tres pilares del desarrollo web: HTML, CSS y JavaScript.

Usos comunes

  • Interactividad en sitios web (animaciones, botones, formularios).

  • Desarrollo front-end (React, Vue, Angular).

  • Desarrollo back-end (Node.js con Express).

  • Aplicaciones móviles (React Native, Ionic).

  • Juegos web, bots y automatización de tareas.

Perro.prototype.constructor = Perro;

    return this.nombre + " ladra";

    funciones.reduceRight((acc, func) => func(acc), valor);

7. Ejemplo Complejo: API REST Client

class APIClient {
    constructor(baseURL) {
        this.baseURL = baseURL;
    }
    
    async get(endpoint) {
        const response = await fetch(`${this.baseURL}${endpoint}`);
        if (!response.ok) throw new Error("Error en la solicitud");
        return await response.json();
    }
    
    async post(endpoint, datos) {
        const response = await fetch(`${this.baseURL}${endpoint}`, {
            method: "POST",
            headers: { "Content-Type": "application/json" },
            body: JSON.stringify(datos)
        });
        return await response.json();
    }
}

const api = new APIClient("https://api.ejemplo.com");
api.get("/usuarios")
    .then(usuarios => console.log(usuarios))
    .catch(error => console.error("Error:", error));

Ejemplo 1: Validador de Formulario

class ValidadorFormulario {
    constructor(formId) {
        this.form = document.getElementById(formId);
        this.errores = {};
    }
    
    validarEmail(email) {
        const regex = /^[^\\s@]+@[^\\s@]+\\.[^\\s@]+$/;
        return regex.test(email);
    }
    
    validarContraseña(password) {
        return password.length >= 8 && /[A-Z]/.test(password);
    }
    
    validar(datos) {
        this.errores = {};
        
        if (!this.validarEmail(datos.email)) {
            this.errores.email = "Email inválido";
        }
        
        if (!this.validarContraseña(datos.password)) {
            this.errores.password = "Contraseña muy débil";
        }
        
        return Object.keys(this.errores).length === 0;
    }
}

// Usar
const validador = new ValidadorFormulario("miFormulario");
const esValido = validador.validar({
    email: "juan@example.com",
    password: "MiPassword123"
});

Ejemplo 2: Carrito de Compras

class Carrito {
    constructor() {
        this.items = [];
    }
    
    agregarProducto(producto, cantidad = 1) {
        const item = this.items.find(i => i.id === producto.id);
        
        if (item) {
            item.cantidad += cantidad;
        } else {
            this.items.push({ ...producto, cantidad });
        }
    }
    
    calcularTotal() {
        return this.items.reduce(
            (total, item) => total + (item.precio * item.cantidad),
            0
        );
    }
    
    obtenerResumen() {
        return {
            cantidadItems: this.items.length,
            total: this.calcularTotal(),
            items: this.items
        };
    }
}

// Usar
const carrito = new Carrito();
carrito.agregarProducto({ id: 1, nombre: "Laptop", precio: 999 });
console.log(carrito.obtenerResumen());

Ejemplo 3: Sistema de Notificaciones

class NotificacionSistema {
    constructor() {
        this.notificaciones = [];
    }
    
    mostrar(mensaje, tipo = "info", duracion = 3000) {
        const id = Date.now();
        const notif = { id, mensaje, tipo, duracion };
        
        this.notificaciones.push(notif);
        
        // Renderizar
        this.renderizar();
        
        // Auto remover
        setTimeout(() => this.remover(id), duracion);
    }
    
    remover(id) {
        this.notificaciones = this.notificaciones
            .filter(n => n.id !== id);
        this.renderizar();
    }
    
    renderizar() {
        const container = document.getElementById("notificaciones");
        container.innerHTML = this.notificaciones.map(n =>
            `<div class="notif ${n.tipo}">${n.mensaje}</div>`
        ).join("");
    }
}

const notificador = new NotificacionSistema();
notificador.mostrar("¡Operación exitosa!", "success");
notificador.mostrar("Ocurrió un error", "error");

Ventajas y Desventajas

Ventajas

  • Muy versátil: cliente y servidor.

  • Funciona en todos los navegadores modernos.

  • Amplia comunidad y miles de recursos y librerías.

  • Esencial para el desarrollo web moderno.

Desventajas

  • La flexibilidad puede llevar a malas prácticas si no se usa con cuidado.

  • Problemas de seguridad si no se gestiona bien.

  • A veces inconsistente entre diferentes navegadores (aunque esto ha mejorado mucho).
