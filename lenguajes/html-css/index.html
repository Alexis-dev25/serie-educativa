HTML & CSS

¿Qué es HTML y CSS?

Introducción

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear páginas web. Define la estructura y el contenido de una página mediante etiquetas semánticas.

CSS (Cascading Style Sheets) es un lenguaje de estilos utilizado para describir la presentación visual de un documento HTML. Controla el diseño, colores, tipografía y efectos visuales.

Juntos, HTML y CSS son los fundamentos del desarrollo web moderno. HTML proporciona la estructura, mientras que CSS proporciona la apariencia y el diseño responsivo.

Más Información

Características principales de HTML

  • Lenguaje de marcado basado en etiquetas (tags).

  • Estructura jerárquica con elementos anidados.

  • Semántica clara para mejorar accesibilidad y SEO.

  • Soporta multimedia: imágenes, video, audio.

  • Formularios interactivos para recopilar datos.

Características principales de CSS

  • Selectores para dirigirse a elementos HTML específicos.

  • Propiedades para cambiar color, tamaño, fuente, posición, etc.

  • Media queries para diseño responsivo (mobile-first).

  • Flexbox y Grid para layouts modernos.

  • Animaciones y transiciones para efectos dinámicos.

1. Selectores CSS

/* Selector de elemento */
h1 { color: blue; }

/* Selector de clase */
.mi-clase { background-color: yellow; }

/* Selector de ID */
#mi-id { font-size: 20px; }

/* Selector de atributo */
input[type="text"] { border: 1px solid gray; }

/* Combinadores */
div > p { margin: 10px; }        /* hijo directo */
div p { color: green; }          /* descendiente */

2. Modelo de Caja (Box Model)

div {
    /* Contenido */
    width: 200px;
    height: 100px;
    
    /* Relleno interno (padding) */
    padding: 20px;
    
    /* Borde */
    border: 2px solid black;
    
    /* Margen externo */
    margin: 30px;
}

/* Shorthand */
padding: 10px 20px;        /* vertical horizontal */
margin: 10px 20px 30px 40px;  /* top right bottom left */

3. Colores y Tipografía

body {
    /* Colores */
    color: #333333;           /* Hexadecimal */
    background-color: rgb(255, 255, 255);
    
    /* Tipografía */
    font-family: Arial, sans-serif;
    font-size: 16px;
    font-weight: bold;        /* bold, 400, 700, etc */
    font-style: italic;
    line-height: 1.5;
    text-align: center;
    text-decoration: underline;
}

/* Fuentes Web */
@font-face {
    font-family: 'MiFuente';
    src: url('fuente.woff2') format('woff2');
}

4. Display y Posicionamiento

/* Display */
.bloque { display: block; }        /* línea completa */
.inline { display: inline; }       /* en línea */
.inline-block { display: inline-block; }
.flex { display: flex; }           /* Flexbox */
.grid { display: grid; }           /* CSS Grid */

/* Posicionamiento */
.relativo { position: relative; top: 10px; left: 20px; }
.absoluto { position: absolute; top: 0; left: 0; }
.fijo { position: fixed; bottom: 20px; right: 20px; }
.estatico { position: static; }    /* por defecto */

1. Flexbox

.contenedor {
    display: flex;
    flex-direction: row;          /* row, column */
    justify-content: center;      /* distribución horizontal */
    align-items: center;          /* distribución vertical */
    gap: 20px;                    /* espacio entre elementos */
}

.item {
    flex: 1;                      /* igual tamaño */
    /* o */
    flex-grow: 1;
    flex-shrink: 1;
    flex-basis: 100px;
}

2. CSS Grid

.grid-contenedor {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: 100px auto 50px;
    gap: 20px;
}

.item {
    grid-column: 1 / 3;           /* del 1 al 3 */
    grid-row: 1 / 2;
}

/* Grid responsivo */
grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));

3. Media Queries (Responsivo)

/* Mobile first */
.contenedor {
    width: 100%;
    font-size: 14px;
}

/* Tablet */
@media (min-width: 768px) {
    .contenedor {
        width: 90%;
        font-size: 16px;
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .contenedor {
        width: 1000px;
        font-size: 18px;
    }
}

/* Orientación */
@media (orientation: landscape) {
    body { flex-direction: row; }
}

4. Variables CSS

:root {
    --color-primario: #007bff;
    --color-secundario: #6c757d;
    --espaciado: 20px;
    --font-principal: Arial, sans-serif;
}

body {
    color: var(--color-primario);
    padding: var(--espaciado);
    font-family: var(--font-principal);
}

/* Variables con fallback */
color: var(--color-no-existe, blue);

5. Transformaciones

div {
    /* Escala */
    transform: scale(1.5);
    
    /* Rotación */
    transform: rotate(45deg);
    
    /* Traslación */
    transform: translate(50px, 30px);
    
    /* Sesgo */
    transform: skew(10deg, 20deg);
    
    /* Combinadas */
    transform: translate(30px) rotate(45deg) scale(1.2);
    
    /* Perspectiva 3D */
    transform: perspective(500px) rotateX(10deg) rotateY(20deg);
}

7. Animaciones Avanzadas

/* Definir animación */
@keyframes deslizar {
    0% { left: 0; opacity: 0; }
    50% { opacity: 1; }
    100% { left: 500px; opacity: 0; }
}

div {
    animation: deslizar 3s ease-in-out infinite;
    animation-delay: 1s;
    animation-iteration-count: 2;
}

/* Transición */
button {
    transition: all 0.3s cubic-bezier(0.25, 0.46, 0.45, 0.94);
    background: linear-gradient(45deg, red, blue);
}
