C#

¿Qué es C#?

Introducción

C# es un lenguaje de programación orientado a objetos moderno, desarrollado por Microsoft en 2000 como parte del framework .NET. Combina la potencia de C++ con la simplicidad de Java.

C# es ampliamente utilizado para desarrollo de aplicaciones Windows, juegos con Unity, aplicaciones web con ASP.NET y aplicaciones móviles multiplataforma.

Más Información

Características principales

  • Lenguaje compilado, fuertemente tipado y orientado a objetos.

  • Soporta programación asincrónica, LINQ y programación funcional.

  • Integrado con .NET Framework y .NET Core (multiplataforma).

  • Soporte para nullable types, pattern matching y records.

  • Excelente para desarrollo de juegos con Unity.

  • Gestión automática de memoria con Garbage Collector.

Usos comunes

  • Desarrollo de juegos con Unity (mayoría de juegos indie).

  • Aplicaciones Windows de escritorio (WPF, WinForms).

  • Aplicaciones web con ASP.NET Core.

  • Aplicaciones móviles multiplataforma con Xamarin.

  • Servicios backend y APIs REST.

Ventajas y Desventajas

Ventajas

  • Sintaxis limpia y moderna, fácil de aprender.

  • Muy versátil: juegos, web, desktop, mobile.

  • Comunidad grande y ecosistema .NET robusto.

  • Excelente documentación y soporte de Microsoft.

Desventajas

  • Históricamente ligado a Windows (aunque ahora multiplataforma).

  • Mayor consumo de memoria comparado con C++.

  • Requiere .NET instalado para ejecutar.

🎯 Ejemplos Complejos

Ejemplo 1: Gestor de Tareas

public class Tarea {
    public string Titulo { get; set; }
    public bool Completada { get; set; }
}

public class GestorTareas {
    private List tareas = new List();
    
    public void Agregar(string titulo) {
        tareas.Add(new Tarea { Titulo = titulo });
    }
    
    public void Completar(int indice) {
        if (indice >= 0 && indice < tareas.Count) {
            tareas[indice].Completada = true;
        }
    }
}

Ejemplo 2: API REST Cliente

public class ApiCliente {
    private HttpClient cliente = new HttpClient();
    
    public async Task ObtenerDatos(string url) {
        try {
            var response = await cliente.GetAsync(url);
            return await response.Content.ReadAsStringAsync();
        } catch (Exception ex) {
            Console.WriteLine("Error: " + ex.Message);
            return null;
        }
    }
}

Ejemplo 3: Sistema de Autenticación

public class Usuario {
    public string Correo { get; set; }
}

public class Autenticador {
    private List usuarios = new List();
    
    public bool Registrar(string correo) {
        if (usuarios.Any(u => u.Correo == correo)) return false;
        usuarios.Add(new Usuario { Correo = correo });
        return true;
    }
}
