C# / CSharp

¿Qué es C#?

Introducción

C# (se pronuncia “C sharp”) es un lenguaje de programación moderno, orientado a objetos, creado por Microsoft en el año 2000 como parte del framework .NET.

  • Fácil de aprender
  • Muy seguro
  • Rápido
  • Flexible
  • Compatible con Windows, Linux, macOS y web

Más Informarción

Características principales

C# se basa en:

  • Clases
  • Objetos
  • Herencia
  • Polimorfismo
  • Encapsulamiento
  • Abstracción

Usos comunes

1. Desarollo de Videojuegos 🎮

  • Es el lenguaje principal del motor Unity (el más usado del mundo junto con Unreal)./li>
  • Ideal para juegos 2D y 3D.

2. Aplicaciones de escritorio 🖥💻

  • Con Windows Forms
  • WPF
  • MAUI (para crear apps nativas modernas)

3. Aplicaciones web 🌐

  • Con ASP.NET Core, uno de los frameworks web más rápidos que existen.

4. Aplicaciones móviles 📱

  • Con Xamarin/Microsoft MAUI puedes crear apps para Android y iOS con un solo código.

5. Inteligencia Artificial y Machine Learning 🤖

  • Con la librería ML.NET

6. Microservicios y servidores de alto rendimiento ⚙

  • Gracias a .NET Core, que es súper eficiente.

Ventajas y Desventajas

Ventajas

  • Es muy rápido ---> .NET Core compite con C++ y Rust en benchmarks de servidores web.
  • Es muy estable ---> Ideal para sistemas grandes.
  • Tiene un ecosistema enorme ---> Miles de librerías, frameworks y herramientas.
  • Gran comunidad ---> Millones de desarrolladores y documentación oficial de calidad.
  • Soporte empresarial ---> Microsoft lleva más de 20 años mejorándolo.

Desventajas

  • Puede ser más complejo al inicio que Python ---> Requiere entender conceptos como clases, métodos, tipos, etc.
  • Para apps móviles necesitas MAUI (aún en desarrollo) ---> Aunque mejora cada año.
  • A veces consume más memoria que C++ o Rust ---> Por el Garbage Collector.
