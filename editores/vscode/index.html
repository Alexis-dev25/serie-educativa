Inicio · Editores · VSCode

Visual Studio Code · Nivel básico

VSCode es un editor ligero, extensible y gratuito, ideal para la mayoría de lenguajes y proyectos.

Nivel: Básico Windows · Linux · macOS Terminal integrada

1. Instalación y configuración

  • Descarga VSCode desde la página oficial.
  • Instálalo con las opciones por defecto.
  • Abre la paleta de comandos con Ctrl+Shift+P (Cmd+Shift+P en macOS).

Extensiones recomendadas

  • Python
  • ESLint / Prettier (para JavaScript)
  • C#
  • GitLens

2. Features clave

  • Explorador de archivos integrado.
  • Terminal integrada (Vista → Terminal).
  • Depuración con puntos de ruptura.
  • Atajos de teclado para editar más rápido.

Puedes abrir tu carpeta de proyectos, lanzar la terminal integrada y seguir los ejemplos de lenguajes desde un solo sitio.
