Android Studio · Nivel avanzado

Android Studio es el entorno oficial para desarrollar aplicaciones Android nativas con Kotlin o Java.

Nivel: Avanzado Requiere buen hardware Windows · Linux · macOS

1. Instalación

  • Descarga Android Studio desde la web oficial.
  • Instálalo con el asistente por defecto.
  • Configura el SDK de Android y crea un emulador (AVD).

Primer proyecto

  • Crea un proyecto "Empty Activity".
  • Selecciona Kotlin como lenguaje recomendado.
  • Ejecuta la app en un emulador o dispositivo físico.

2. Flujo de trabajo

  • Diseño de interfaces con XML / Compose.
  • Uso del logcat para depurar.
  • Gestión de dependencias con Gradle.

Antes de llegar a Android Studio, es recomendable que domines al menos un lenguaje (Kotlin/Java) y conceptos básicos de programación.
