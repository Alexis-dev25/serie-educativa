¿Qué es Sublime Text?

Sublime Text es un editor de código ligero, muy rápido y altamente personalizable. Favorito de muchos desarrolladores por su velocidad y flexibilidad.

Características principales:

  • ⚡ Extremadamente rápido y ligero
  • 🎨 Altamente personalizable
  • 📦 Amplio ecosistema de plugins (Package Control)
  • 🔍 Búsqueda y reemplazo potente
  • 🖥️ Multi-plataforma (Windows, Mac, Linux)
  • 💰 Licencia de prueba indefinida

Instalación

# macOS (Homebrew)
brew install --cask sublime-text

# Windows (Chocolatey)
choco install sublimetext4

# Linux (Ubuntu/Debian)
sudo apt install sublime-text

# O descargar desde: https://www.sublimetext.com/download

Primeros pasos

Instalar Package Control (IMPORTANTE):

Package Control es el gestor de plugins de Sublime Text.

  1. Abre Sublime Text
  2. View → Show Console
  3. Copia y pega: import urllib.request,os,hashlib; h = 'df21e130d211cee9530fff8346c13c16' + 'e1fb0c1c0426ca88d8205e1c36423c7d'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else __import__('shutil').copyfile( os.path.join( ipp, pf) if os.path.exists( os.path.join( ipp, pf)) else urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(), os.path.join( ipp, pf)) ; print('Please restart Sublime Text 3')
  4. Presiona Enter y reinicia Sublime Text
  5. Verifica: Preferences → Package Control

Plugins esenciales (con Package Control)

Pasos para instalar:

  1. Ctrl+Shift+P (Cmd+Shift+P en Mac)
  2. Escribe "Package Control: Install Package"
  3. Busca el plugin deseado y presiona Enter

💡 Plugins recomendados:

  • Emmet - Autocomplete HTML/CSS avanzado
  • Prettier - Formateador de código
  • Pylint - Linting para Python
  • ESLint - Linting para JavaScript
  • GitGutter - Ver cambios de Git
  • Color Highlighter - Resalta colores
  • Bracket Highlighter - Destaca brackets pareados
  • SideBarEnhancements - Mejoras al sidebar
  • A File Icon - Iconos para archivos en el sidebar
  • Dracula Color Scheme - Tema popular

Build Systems (Ejecutar código)

Crear build system para Python:

  1. Tools → Build System → New Build System
  2. Copia esto:
{
    "cmd": ["python", "-u", "$file"],
    "file_regex": "^[ ]*File \"(...*?)\", line ([0-9]*)",
    "selector": "source.python"
}
  1. Guarda como "Python.sublime-build"
  2. Ahora puedes presionar Ctrl+B para ejecutar Python

Snippets personalizados

Crear snippet:

  1. Tools → Developer → New Snippet
  2. Ejemplo para HTML boilerplate:
<snippet>
    <content><![CDATA[
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>${1:Título}</title>
</head>
<body>
    ${2:Contenido}
</body>
</html>
]]></content>
    <tabTrigger>html5</tabTrigger>
    <scope>text.html</scope>
    <description>HTML5 Boilerplate</description>
</snippet>
  1. Guarda como "HTML5.sublime-snippet"
  2. Ahora escribe "html5" + Tab para expandir

Tips avanzados

Multi-cursor:

  • Ctrl+Click (Cmd+Click en Mac) para múltiples cursores
  • Ctrl+Alt+↑/↓ para duplicar cursor arriba/abajo
  • Ctrl+D selecciona palabra, repite para multiples
