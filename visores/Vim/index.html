¿Qué es Vim?

Vim (Vi IMproved) es un editor de texto extremadamente potente que funciona en la terminal. Es el favorito de programadores y sysadmins por su eficiencia.

Características:

  • ⚡ Ultrarrápido y ligero
  • 🎯 Basado en modos (mejora productividad)
  • 🔌 Completamente personalizable
  • 🖥️ Funciona en cualquier terminal/servidor
  • 🧠 Curva de aprendizaje pero muy rewarding
  • 🎨 Amplio soporte de temas y plugins

¿Por qué aprender Vim?

  • ✅ Editar archivos en servidores remotos (SSH)
  • ✅ Desarrollo sin abandonar la terminal
  • ✅ Atajos que duplican tu velocidad de edición
  • ✅ Funciona en 40 años de sistemas UNIX
  • ✅ Lenguaje de edición (composable)

Instalación

# Linux (Ubuntu/Debian)
sudo apt install vim vim-gui-common

# macOS (Homebrew)
brew install vim

# O usar Vi que está preinstalado
vi nombre-archivo.txt

# Verificar versión
vim --version

Los 4 Modos Principales

1. NORMAL (Navegación y edición)

El modo por defecto. Aquí escribes comandos para navegar y editar.

Cómo entrar: Presiona ESC desde cualquier modo

2. INSERT (Insertar texto)

Aquí escribes el código normalmente, como en un editor convencional.

Cómo entrar: Presiona i, a, o, I, A o O desde NORMAL

3. COMMAND LINE (Comandos Ex)

Ejecutas comandos más complejos (guardar, buscar y reemplazar, etc.)

Cómo entrar: Presiona : desde NORMAL

4. VISUAL (Selección)

Selecciona texto para aplicar operaciones.

Cómo entrar: Presiona v, V o Ctrl+V desde NORMAL

Comandos básicos (Modo NORMAL)

Movimiento (hjkl)

h              # Izquierda
j              # Abajo
k              # Arriba
l              # Derecha

w              # Siguiente palabra
b              # Palabra anterior
e              # Fin de palabra
0              # Inicio de línea
$              # Fin de línea
G              # Fin del archivo
gg             # Inicio del archivo
:20            # Ir a línea 20

Edición básica

i              # Insertar antes del cursor
a              # Insertar después del cursor
o              # Nueva línea abajo e insertar
O              # Nueva línea arriba e insertar
x              # Eliminar carácter
dd             # Eliminar línea
d5d            # Eliminar 5 líneas
dw             # Eliminar palabra
d$             # Eliminar hasta fin de línea
u              # Deshacer
Ctrl+r         # Rehacer
.              # Repetir último comando

Copia y pega

yy             # Copiar línea completa
5yy            # Copiar 5 líneas
yw             # Copiar palabra
p              # Pegar después del cursor
P              # Pegar antes del cursor
v              # Seleccionar (VISUAL mode)
y              # Copiar selección
d              # Cortar selección

Búsqueda y reemplazo

/texto         # Buscar texto
n              # Siguiente coincidencia
N              # Coincidencia anterior
:s/viejo/nuevo/    # Reemplazar en línea actual
:%s/viejo/nuevo/   # Reemplazar en todo el archivo
:%s/viejo/nuevo/gc # Reemplazar con confirmación

Modo de comando (:)

:w             # Guardar
:q             # Salir
:wq            # Guardar y salir
:q!            # Salir sin guardar
:e archivo.txt # Abrir archivo
:split         # Dividir ventana horizontalmente
:vsplit        # Dividir ventana verticalmente
:set number    # Mostrar números de línea
:set ignorecase # Búsqueda sin mayúsculas

Comandos avanzados

Operadores (composables)

La potencia de Vim: puedes combinar operadores con movimientos.

d + w = dw     # Eliminar palabra
c + i + " = ci"  # Cambiar dentro de comillas
y + t + , = yt,  # Copiar hasta coma
> + > + j = >>j  # Indentar dos líneas

Macros

q + a          # Empezar a grabar en registro 'a'
(tus comandos...)
q              # Parar grabación
@a             # Ejecutar macro en 'a'
5@a            # Ejecutar macro 5 veces

Selección visual avanzada

v              # Visual (por carácter)
V              # Visual (línea completa)
Ctrl+v         # Visual bloque (rectangular)
gv             # Reseleccionar última selección
:<,> comando  # Aplicar comando a selección

Búsqueda y navegación avanzada

*              # Buscar palabra bajo cursor
#              # Buscar palabra anterior
%              # Ir a bracket pareado
[[ ]]          # Navegar por funciones
{ }            # Navegar por párrafos
:!comando      # Ejecutar comando shell

Configuración (~/.vimrc)

Crear archivo de configuración:

" Configuración básica recomendada
set number              " Mostrar números de línea
set relativenumber      " Números relativos (mejor para movimientos)
set tabstop=4           " Ancho de tabulación
set shiftwidth=4        " Ancho de indentación
set expandtab           " Convertir tabs a espacios
set autoindent          " Autoindentación
set ignorecase          " Búsqueda sin mayúsculas
set smartcase           " Pero si escribes mayúsculas, sí considera
set incsearch           " Búsqueda incremental
set hlsearch            " Resaltar búsquedas
set mouse=a             " Habilitar mouse
syntax on               " Resaltado de sintaxis
colorscheme desert      " Tema (prueba: slate, monokai, dracula)

" Mapeos personalizados
map <leader>n :NERDTreeToggle<CR>    " Toggle NERDTree
let mapleader = ","     " Definir líder como ','

Plugins con vim-plug

Instalar vim-plug:

curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

Configurar plugins en ~/.vimrc:

call plug#begin('~/.vim/plugged')

Plug 'tpope/vim-sensible'           " Configuración sensible
Plug 'tpope/vim-surround'           " Manipular delimitadores
Plug 'scrooloose/nerdtree'          " Explorador de archivos
Plug 'junegunn/fzf'                 " Búsqueda fuzzy
Plug 'vim-airline/vim-airline'      " Barra de estado hermosa
Plug 'dracula/vim'                  " Tema Dracula
Plug 'sheerun/vim-polyglot'         " Soporte de lenguajes

call plug#end()

" Ahora en Vim: :PlugInstall

Plugins recomendados:

  • vim-surround - Cambiar delimitadores rápido
  • vim-commentary - Comentar código
  • NERDTree - Explorador de archivos
  • fzf.vim - Búsqueda fuzzy (buscar archivos rápido)
  • CoC - Autocompletado inteligente
  • vim-airline - Barra de estado hermosa
  • dracula/vim - Tema oscuro popular

Flujo de trabajo típico

# 1. Abrir archivo
vim archivo.py

# 2. Entrar a insert mode y escribir
i (presionar i)
(escribir código...)

# 3. Volver a normal mode y guardar
ESC (presionar ESC)
:wq (presionar :wq y Enter)

Diferencia: Vim vs Vi vs Neovim

Editor Descripción Ventaja
Vi Original (1976), preinstalado en todos los Unix Siempre disponible en servidores antiguos
Vim Vi Improved (1991), mucho más potente Mejor compatibilidad, más plugins, más activo
Neovim Refactor moderno de Vim (2014) Mejor arquitectura, plugins asíncronos, Lua scripting
