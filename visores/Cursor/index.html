¿Qué es Cursor?

Cursor es un editor de código moderno basado en VS Code pero potenciado con IA integrada. Diseñado para acelerar el desarrollo usando autocompletado inteligente y asistencia de programación.

Características revolucionarias:

  • 🤖 IA integrada (GPT-4, Claude, Copilot+)
  • ⚡ Autocompletado inteligente en tiempo real
  • 💬 Chat integrado para preguntas sobre código
  • 🔄 Refactorización automática propuesta por IA
  • 🐛 Debugging asistido por IA
  • 📝 Generación de código desde comentarios
  • 👨‍💻 Compatible con extensiones de VS Code
  • ⌨️ Mismos atajos que VS Code

¿Cuándo usar Cursor?

  • ✅ Desarrollo rápido de prototipos
  • ✅ Aprender nuevos lenguajes/frameworks
  • ✅ Generar código boilerplate
  • ✅ Refactorización de código
  • ✅ Debugging asistido
  • ✅ Documentación automática

Configuración inicial

1. Autenticación con IA

Al abrir Cursor por primera vez:

  1. Te pedirá que selecciones un modelo de IA
  2. Opciones: GPT-4, Claude, Copilot+ (según suscripción)
  3. Inicia sesión o conecta tu cuenta
  4. Acepta los términos y condiciones

2. Sincronizar extensiones de VS Code

  1. File → Preferences → Settings
  2. Busca "Cursor: Import VS Code Extensions"
  3. Importará automáticamente tus extensiones favoritas

3. Elegir modelo de IA predeterminado

  1. Cursor Settings (abajo a la izquierda)
  2. Selecciona "Models"
  3. Elige tu modelo preferido para cada tarea

Características principales de IA

💬 Chat (Cmd/Ctrl + K)

Chat integrado con contexto de tu proyecto. Formula preguntas sobre código, pide refactorización o explicaciones.

// En el editor, presiona Ctrl+K
// Escribe tu pregunta:
"¿Cómo optimizar esta función?"
"Refactoriza este código con mejor practices"
"Explica qué hace esta función"

✨ Autocompletado Inteligente (Tab)

Sugiere líneas o bloques de código completos basado en contexto.

  • Empieza a escribir código
  • Cursor sugiere líneas/bloques automáticamente
  • Presiona Tab para aceptar
  • Presiona Escape para rechazar

📝 Generar código desde comentarios

Escribe un comentario descriptivo y Cursor genera el código.

// Función que calcula fibonacci recursivo con cache
// Cursor generará:
def fibonacci(n, cache=None):
    if cache is None:
        cache = {}
    if n in cache:
        return cache[n]
    if n <= 1:
        return n
    cache[n] = fibonacci(n-1, cache) + fibonacci(n-2, cache)
    return cache[n]

🔄 Refactorizar (Cmd/Ctrl + Shift + R)

Selecciona código y Cursor propone mejoras automáticas.

  1. Selecciona código que quieres refactorizar
  2. Presiona Cmd/Ctrl + Shift + R
  3. Cursor propone versión mejorada
  4. Acepta o rechaza cambios

🐛 Debugging Asistido

Cursor entiende errores y propone soluciones.

  1. Cuando hay error, Cursor lo destaca
  2. Hover sobre error para ver explicación
  3. Cursor sugiere fixes automáticos
  4. Acepta sugerencia o ignora

📚 Documentación Automática

Genera docstrings y comentarios automáticamente.

// Coloca cursor en función sin documentar
// Presiona Cmd/Ctrl + K
// Escribe: "Documenta esta función"
// Cursor genera documentación completa

Casos de uso prácticos

📱 Crear una API REST rápidamente

  1. Crea archivo main.py
  2. Escribe comentario:
# API REST con FastAPI que gestiona usuarios
# GET /users - obtener todos los usuarios
# POST /users - crear nuevo usuario
# GET /users/{id} - obtener usuario por ID
# DELETE /users/{id} - eliminar usuario

# Presiona Cmd/Ctrl + K → "Genera el código"
# Cursor generará la API completa

🐍 Convertir código a otro lenguaje

  1. Selecciona código en Python
  2. Chat (Cmd/Ctrl + K): "Convierte esto a JavaScript"
  3. Cursor traduce el código automáticamente

🔍 Entender código legacy

  1. Selecciona código viejo que no entiendes
  2. Chat: "¿Qué hace esta función? Explica en párrafos"
  3. Cursor explica en lenguaje natural

✅ Tests automáticos

  1. Selecciona función
  2. Chat: "Genera tests unitarios completos"
  3. Cursor crea tests automáticamente

🎨 Generar HTML/CSS desde descripción

// Crea un archivo index.html vacío
// Escribe en comentario:
// Un formulario de login moderno con fondo gradiente,
// input de email y password, botón submit azul

// Presiona Cmd/Ctrl + K → "Genera el HTML y CSS"
// Cursor crea un formulario hermoso

Configuración avanzada

Personalizar modelo por lenguaje:

// En settings.json:
{
    "cursor.chat.model": "gpt-4",
    "cursor.autocomplete.model": "gpt-3.5-turbo",
    "[python]": {
        "cursor.chat.model": "claude-3"
    },
    "[javascript]": {
        "cursor.chat.model": "gpt-4"
    }
}

Contexto del proyecto (.cursorrules)

Crea archivo `.cursorrules` en raíz del proyecto para dar contexto a IA:

# .cursorrules
You are helping with a Python Django project.
Use Django ORM for database queries.
Follow PEP 8 style guide.
Add type hints to all functions.
Write docstrings for all functions.
Prioritize performance and security.
Use Django REST Framework for APIs.
Create tests for all new code.

Usar @selecciones en Chat

En Chat, puedes referencias archivos y contexto:

// En chat, escribe:
@models.py - "¿Qué campos tiene el modelo User?"
@utils.js - "Refactoriza esta función siguiendo best practices"
#select - Usa selección actual como contexto

Tips para máximo provecho

💡 Mejores prácticas:

  • ✅ Escribe comentarios descriptivos para mejor contexto
  • ✅ Usa `.cursorrules` para guiar el comportamiento de IA
  • ✅ Selecciona código específico para evitar cambios innecesarios
  • ✅ Usa modelo GPT-4 para tareas complejas, GPT-3.5 para rápidas
  • ✅ Revisa SIEMPRE el código generado antes de usar

⚠️ Cuidados:

  • ⚠️ Cursor no reemplaza aprendizaje, es herramienta
  • ⚠️ Código generado puede tener bugs, necesita testing
  • ⚠️ Ten cuidado con datos sensibles en chat
  • ⚠️ Siempre revisa security implications del código
