¿Qué es Visual Studio Code?

VS Code es un editor de código ligero, potente y gratuito creado por Microsoft. Es el editor más popular entre desarrolladores actualmente.

Características principales:

  • ⚡ Muy rápido y responsivo
  • 🧪 Debugging integrado
  • 🔧 Terminal integrada
  • 🎨 Miles de temas y extensiones
  • 🔍 IntelliSense (autocompletado inteligente)
  • 📦 Git integrado
  • 👥 Live Share (colaborar en tiempo real)
  • ☁️ Cloud Sync (sincronizar configuración en nube)

Instalación

# macOS (Homebrew)
brew install --cask visual-studio-code

# Windows (Chocolatey)
choco install vscode

# Linux (Ubuntu/Debian)
sudo apt install code

# O descargar desde: https://code.visualstudio.com

Extensiones imprescindibles

Pasos para instalar extensiones:

  1. Presiona Ctrl+Shift+X (Cmd+Shift+X en Mac)
  2. Busca la extensión deseada
  3. Presiona "Install"

🌟 Extensiones esenciales:

  • Prettier - Formateador de código automático
  • Pylance - IntelliSense mejorado para Python
  • ESLint - Linting para JavaScript
  • Thunder Client - Cliente REST (como Postman)
  • Live Server - Servidor web local con reload automático
  • GitLens - Información de Git integrada
  • Bracket Pair Colorizer 2 - Destaca brackets pareados
  • Better Comments - Comentarios con colores
  • Path Intellisense - Autocompletado de rutas
  • Code Runner - Ejecutar código rápidamente
  • REST Client - Realizar requests HTTP desde VS Code
  • Remote - SSH - Editar archivos en servidores remotos

🎨 Temas populares:

  • Dracula Official - Tema oscuro muy popular
  • One Dark Pro - Inspirado en Atom
  • Nord - Colores árticos relajantes
  • Ayu - Minimalista y limpio
  • Gruvbox Dark - Colores retro cálidos
  • Synthwave 84 - Estilo cyberpunk

📦 Extensiones por lenguaje:

  • Python: Pylance, Python, Jupyter
  • JavaScript/Node: ESLint, Prettier, Thunder Client

Configuración (settings.json)

Acceder a configuración:

File → Preferences → Settings (Ctrl+,)

O editar directamente: Ctrl+Shift+P → Preferences: Open Settings (JSON)

Configuración recomendada:

{
    "editor.fontSize": 14,
    "editor.fontFamily": "Fira Code, Consolas",
    "editor.formatOnSave": true,
    "editor.defaultFormatter": "esbenp.prettier-vscode",
    "editor.tabSize": 2,
    "editor.insertSpaces": true,
    "editor.wordWrap": "on",
    "editor.rulers": [80, 120],
    "editor.minimap.enabled": false,
    "files.autoSave": "afterDelay",
    "workbench.colorTheme": "Dracula Official",
    "workbench.iconTheme": "material-icon-theme",
    "python.linting.enabled": true,
    "python.linting.pylintEnabled": true,
    "[python]": {
        "editor.defaultFormatter": "ms-python.python",
        "editor.formatOnSave": true
    },
    "[javascript]": {
        "editor.defaultFormatter": "esbenp.prettier-vscode"
    },
    "git.autofetch": true,
    "terminal.integrated.shell.windows": "C:\\\\Program Files\\\\PowerShell\\\\7\\\\pwsh.exe"
}

Debugging (Depuración)

Debugging de Python:

  1. Instala extensión "Python" o "Pylance"
  2. Crea archivo .vscode/launch.json:
{
    "version": "0.2.0",
    "configurations": [
        {
            "name": "Python: Current File",
            "type": "python",
            "request": "launch",
            "program": "${file}",
            "console": "integratedTerminal"
        }
    ]
}
  1. Presiona F5 para iniciar depuración
  2. Usa breakpoints (Ctrl+Click en línea) para parar ejecución

Debugging de Node.js:

{
    "version": "0.2.0",
    "configurations": [
        {
            "type": "node",
            "request": "launch",
            "name": "Launch Program",
            "program": "${workspaceFolder}/app.js",
            "skipFiles": ["<node_internals>/**"]
        }
    ]
}

Características avanzadas

Multi-cursor y edición:

  • Ctrl+D - Selecciona palabra siguiente (usa múltiples veces)
  • Ctrl+Shift+L - Selecciona todas las instancias
  • Ctrl+Alt+↑/↓ - Duplica línea arriba/abajo
  • Shift+Alt+↑/↓ - Mueve línea arriba/abajo

Snippets personalizados:

  1. File → Preferences → User Snippets
  2. Selecciona lenguaje (ej: "python")
  3. Ejemplo:
{
    "Print Function": {
        "prefix": "pf",
        "body": [
            "print(f\"${1:variable} = {${1:variable}}\")"
        ],
        "description": "Print f-string debug"
    }
}

Workspaces (Espacios de trabajo):

Guarda múltiples proyectos juntos:

  1. File → Save Workspace As...
  2. VS Code guardará archivo .code-workspace
  3. Abre con: File → Open Workspace from File

Live Share (Colaboración):

  1. Instala extensión "Live Share"
  2. Presiona Ctrl+Shift+P → "Live Share: Start Collaboration Session"
  3. Comparte link con compañeros para colaborar en tiempo real

Terminal integrada

Usar terminal:

  • Abre: Ctrl+` (acento grave)
  • Nueva terminal: Ctrl+Shift+`
  • Cambiar shell: Ctrl+Shift+P → "Terminal: Select Default Profile"

Ventajas:

  • No necesitas dejar VS Code
  • La terminal hereda variables del workspace
  • Puedes tener múltiples terminales abiertas
  • Output resaltado por sintaxis en Python, JavaScript, etc.

Control de versiones (Git)

Git integrado:

  • Ctrl+Shift+G - Abre panel de Git
  • Ver cambios sin salir de VS Code
  • Commit, push, pull desde la interfaz
  • Resolve merge conflicts visualmente

Recomendación: Instala GitLens

Extensión que muestra quién cambió cada línea, historial de commits, y más.

Performance tips

Optimizar VS Code:

  • ⚡ Desactiva extensiones innecesarias
  • 🖥️ Aumenta memoria: launch.json
  • 📁 Excluye carpetas de búsqueda: settings.json
  • 🔍 Usa .gitignore para ignorar archivos
// Excluir carpetas
"files.exclude": {
    "**/.git": true,
    "**/node_modules": true,
    "**/.venv": true
}

// Excluir de búsqueda
"search.exclude": {
    "**/node_modules": true,
    "**/.git": true
}
