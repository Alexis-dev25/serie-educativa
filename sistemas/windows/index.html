Windows para programar · Nivel básico

Prepara Windows como entorno cómodo para programar con Python, JavaScript, C# y otros lenguajes.

Nivel: Básico PowerShell y WSL Recomendado para empezar

1. Terminales en Windows

  • PowerShell (incluido por defecto).
  • Windows Terminal (interfaz moderna con pestañas).
  • WSL (Subsistema de Windows para Linux).

Instalar WSL (opcional)

  • Abre PowerShell como administrador.
  • Ejecuta wsl --install y sigue las instrucciones.
  • Elige una distribución (ej: Ubuntu) y configura tu usuario.

2. Buenas prácticas en Windows

  • Evita rutas con espacios o tildes en proyectos.
  • Activa la extensión de archivos en el explorador.
  • Utiliza un directorio de trabajo como C:\dev.
  • Instala Git y conéctalo con tu editor.

Muchos tutoriales asumen Linux; con WSL puedes seguirlos casi tal cual desde Windows.
