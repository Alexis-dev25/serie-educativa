Linux para desarrolladores · Nivel intermedio

Linux es el sistema de referencia en servidores y un entorno muy potente para programar usando la terminal.

Nivel: Intermedio Distribuciones: Ubuntu, Debian, Fedora... Ideal para back-end

1. Gestor de paquetes

  • Ubuntu/Debian: apt.
  • Fedora: dnf.
  • Arch: pacman.

Ejemplos

  • sudo apt update && sudo apt upgrade
  • sudo apt install python3 python3-pip git
  • sudo apt install nodejs npm

2. Conceptos clave

  • Permisos de archivos (lectura, escritura, ejecución).
  • Usuarios, grupos y sudo.
  • Rutas absolutas y relativas.
  • Uso de la terminal como herramienta principal.

La terminal simulada de la página principal está inspirada en el comportamiento de shells como Bash, pero funciona de forma segura en tu navegador.
