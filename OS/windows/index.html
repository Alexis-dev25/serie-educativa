¿Qué es Windows?

Windows es el sistema operativo más popular del mundo, desarrollado por Microsoft. Domina en computadoras de escritorio y portátiles.

Versiones actuales:

  • Windows 11: Última versión con interfaz moderna
  • Windows 10: Aún muy usado, soporte extendido
  • Windows Server: Para servidores empresariales

🖥️ CMD (Símbolo del Sistema)

La interfaz clásica de línea de comandos de Windows.

Comandos básicos:

dir                    # Listar archivos y carpetas
cd carpeta            # Cambiar de directorio
cd ..                 # Ir al directorio superior
mkdir nombre          # Crear carpeta
del archivo.txt       # Eliminar archivo
copy archivo.txt copia.txt   # Copiar archivo
move archivo.txt destino/    # Mover archivo
echo "texto" > archivo.txt   # Crear archivo con texto
type archivo.txt      # Ver contenido de archivo
cls                   # Limpiar pantalla

Comandos útiles:

ipconfig              # Ver configuración de red
ping google.com       # Verificar conectividad
tasklist              # Listar procesos activos
taskkill /IM proceso.exe   # Terminar proceso
systeminfo            # Información del sistema
date                  # Mostrar fecha
time                  # Mostrar hora

⚡ PowerShell

Shell más potente y moderno que CMD. Basado en .NET, ideal para automatización.

Comandos básicos (Cmdlets):

Get-ChildItem         # Listar archivos (equivalente a dir)
Get-Location          # Ubicación actual (pwd)
Set-Location ruta     # Cambiar directorio (cd)
New-Item -Type Directory -Name carpeta   # Crear carpeta
Remove-Item archivo.txt   # Eliminar archivo
Copy-Item archivo.txt copia.txt  # Copiar archivo
Move-Item archivo.txt destino/   # Mover archivo
Get-Content archivo.txt   # Ver contenido de archivo
Clear-Host            # Limpiar pantalla

Scripting básico:

# Variables
$nombre = "Juan"
$edad = 25
Write-Host "Hola $nombre, tienes $edad años"

# Array
$frutas = "manzana", "plátano", "naranja"
$frutas | ForEach-Object { Write-Host $_ }

# Condicional
if ($edad -gt 18) {
    Write-Host "Es mayor de edad"
} else {
    Write-Host "Es menor de edad"
}

# Bucle
for ($i = 1; $i -le 5; $i++) {
    Write-Host "Número: $i"
}

Funciones en PowerShell:

function Saludar ($nombre) {
    Write-Host "Hola, $nombre!"
}

Saludar "María"

# Función con retorno
function Suma ($a, $b) {
    return $a + $b
}

$resultado = Suma 5 3
Write-Host "5 + 3 = $resultado"

💻 Desarrollo en Windows

Instalación de herramientas:

# Node.js
# Descargar desde nodejs.org o usar chocolatey:
choco install nodejs

# Python
choco install python

# Git
choco install git

# Visual Studio Code
choco install vscode

# .NET SDK (para C#)
choco install dotnet-sdk

Gestores de paquetes en Windows:

  • Chocolatey: Gestor de paquetes para Windows
  • Windows Package Manager (winget): Oficial de Microsoft
  • NPM: Para paquetes JavaScript/Node.js
  • pip: Para paquetes Python
  • NuGet: Para paquetes .NET

Configurar PATH:

# En PowerShell para añadir a PATH:
$env:PATH += ";C:\Program Files\tu_programa"

# Permanentemente editar variables de entorno:
# 1. Win + X → Sistema
# 2. Variables de entorno
# 3. Editar las variables del sistema

WSL (Windows Subsystem for Linux):

Ejecuta Linux directamente en Windows para mejor experiencia de desarrollo.

# Instalar WSL2:
wsl --install

# Instalar distribución específica:
wsl --install -d Ubuntu

# Listar distribuciones:
wsl -l -v

# Entrar en WSL:
wsl

🔧 Administración

Gestión de usuarios:

# PowerShell como administrador
Get-LocalUser          # Listar usuarios
New-LocalUser -Name usuario -Password (ConvertTo-SecureString "contraseña" -AsPlainText -Force)
Remove-LocalUser -Name usuario
Add-LocalGroupMember -Group "Administradores" -Member usuario

Gestión de servicios:

Get-Service                # Listar servicios
Start-Service -Name Spooler    # Iniciar servicio
Stop-Service -Name Spooler     # Detener servicio
Restart-Service -Name Spooler  # Reiniciar servicio

Monitoreo del sistema:

Get-Process            # Ver procesos activos
Get-Process | Sort-Object WorkingSet | Select-Object -Last 5   # Top 5 procesos por memoria
Get-WmiObject Win32_OperatingSystem | Select-Object Caption, Version, OSArchitecture
Get-Volume             # Ver unidades de disco

Mantenimiento:

# Limpiar disco
Disk Cleanup           # Utilidad gráfica
del C:\Windows\Temp\*  # Limpiar carpeta temporal

# Desfragmentar disco
Defrag-Volume -DriveLetter C -Defrag

# Actualizar Windows
Get-WindowsUpdate
Install-WindowsUpdate
