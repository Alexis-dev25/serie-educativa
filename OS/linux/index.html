¿Qué es Linux?

Linux es un sistema operativo de código abierto, gratuito y potente. Domina en servidores, supercomputadoras y dispositivos embebidos.

Distribuciones populares:

  • Ubuntu: La más popular y amigable para principiantes
  • Debian: Estable y confiable, base de muchas otras
  • Red Hat/Fedora: Profesional, usada en empresas
  • Arch Linux: Minimalista, para usuarios avanzados
  • CentOS: Basada en Red Hat, ideal para servidores

¿Por qué Linux es popular?

  • ✅ Código abierto y gratuito
  • ✅ Muy seguro y estable
  • ✅ Poderosa línea de comandos
  • ✅ Soporte de comunidad activa
  • ✅ Ideal para servidores y desarrollo

🖤 Bash Shell

El intérprete de comandos más común en Linux.

Comandos básicos:

ls                     # Listar archivos
ls -la                 # Lista detallada con archivos ocultos
pwd                    # Mostrar directorio actual
cd directorio          # Cambiar directorio
cd ..                  # Ir al directorio superior
cd ~                   # Ir al home
mkdir nombre           # Crear directorio
rmdir nombre           # Eliminar directorio vacío
rm archivo.txt         # Eliminar archivo
rm -r carpeta/         # Eliminar carpeta y contenido
cp archivo.txt copia.txt    # Copiar archivo
mv archivo.txt nuevo_nombre # Mover/renombrar archivo

Visualizar contenido:

cat archivo.txt        # Ver contenido completo
less archivo.txt       # Ver con scroll (q para salir)
head -n 20 archivo.txt # Ver primeras 20 líneas
tail -n 10 archivo.txt # Ver últimas 10 líneas
grep "texto" archivo.txt   # Buscar texto en archivo
wc -l archivo.txt      # Contar líneas

Permisos:

chmod 755 script.sh    # Cambiar permisos (rwxr-xr-x)
chmod +x script.sh     # Hacer ejecutable
chown usuario archivo.txt  # Cambiar propietario
ls -l                  # Ver permisos detallados

# Permisos: r(4) w(2) x(1)
# 755 = rwxr-xr-x
# 644 = rw-r--r--
# 777 = rwxrwxrwx

Scripting Bash:

#!/bin/bash
# Variables
nombre="Juan"
edad=25
echo "Hola $nombre"

# Array
frutas=("manzana" "plátano" "naranja")
echo ${frutas[0]}

# Condicional
if [ $edad -gt 18 ]; then
    echo "Es mayor de edad"
else
    echo "Es menor de edad"
fi

# Bucle for
for fruta in "${frutas[@]}"; do
    echo "Fruta: $fruta"
done

# Bucle while
contador=1
while [ $contador -le 5 ]; do
    echo "Número: $contador"
    contador=$((contador + 1))
done

# Función
saludar() {
    echo "Hola, $1!"
}

saludar "María"

📁 Sistema de Archivos

Estructura de directorios:

/              # Raíz del sistema de archivos
/home          # Directorios de usuarios
/etc           # Archivos de configuración
/var           # Datos variables (logs, caches)
/usr           # Programas y librerías de usuario
/bin           # Comandos esenciales
/lib           # Librerías del sistema
/tmp           # Archivos temporales
/root          # Home del usuario root
/opt           # Programas opcionales

Navegación y búsqueda:

find /ruta -name "archivo.txt"    # Buscar archivo
find /ruta -type f -name "*.js"   # Buscar archivos .js
locate archivo.txt                # Búsqueda rápida (requiere indexación)
which comando                      # Ubicación del comando
whereis comando                    # Ubicación de comando y documentación

# Aliases (accesos cortos)
alias ll='ls -la'
alias backup='cp -r /home/datos /backup/'

Operaciones con archivos:

touch archivo.txt      # Crear archivo vacío
echo "texto" > archivo.txt   # Crear archivo con contenido
echo "más texto" >> archivo.txt   # Añadir al archivo

# Comprimido
tar -czf archivo.tar.gz directorio/   # Comprimir
tar -xzf archivo.tar.gz              # Descomprimir

# Disco
du -sh directorio/     # Tamaño de directorio
df -h                  # Espacio disponible en discos

📦 Gestión de Paquetes

APT (Ubuntu/Debian):

sudo apt update        # Actualizar lista de paquetes
sudo apt upgrade       # Actualizar paquetes instalados
sudo apt install nombre-paquete    # Instalar paquete
sudo apt remove nombre-paquete     # Desinstalar paquete
sudo apt search node   # Buscar paquete
apt list --installed   # Listar paquetes instalados
sudo apt autoremove    # Limpiar paquetes no usados

YUM/DNF (Red Hat/Fedora):

sudo dnf install nombre-paquete   # Instalar
sudo dnf remove nombre-paquete    # Desinstalar
sudo dnf update                   # Actualizar
sudo dnf search paquete           # Buscar
dnf list installed                # Listar instalados

Instalación desde fuente:

tar -xzf programa.tar.gz
cd programa/
./configure
make
sudo make install

# Desinstalar
sudo make uninstall

Herramientas de desarrollo:

sudo apt install build-essential   # Compiladores C/C++
sudo apt install python3-dev       # Python desarrollo
sudo apt install nodejs npm        # Node.js y NPM
sudo apt install git               # Git control de versiones
sudo apt install curl wget         # Descargar archivos
sudo apt install vim emacs         # Editores de texto

🔧 Administración del Sistema

Usuarios y permisos:

sudo su                # Convertirse en root
sudo -l                # Ver comandos permitidos con sudo
useradd nuevo-usuario  # Crear usuario (como root)
passwd usuario         # Cambiar contraseña
userdel usuario        # Eliminar usuario

# Grupos
groupadd grupo-nuevo
usermod -a -G grupo-nuevo usuario
id usuario             # Ver grupos del usuario

Procesos:

ps aux                 # Listar todos los procesos
ps aux | grep nombre   # Buscar proceso específico
top                    # Monitoreo en tiempo real
htop                   # Top mejorado (más amigable)
kill PID               # Terminar proceso por PID
kill -9 PID            # Forzar terminar proceso
bg                     # Ejecutar en background
fg                     # Traer al foreground
jobs                   # Ver trabajos en background

Servicios (systemd):

sudo systemctl start nombre-servicio   # Iniciar
sudo systemctl stop nombre-servicio    # Detener
sudo systemctl restart nombre-servicio # Reiniciar
sudo systemctl status nombre-servicio  # Ver estado
sudo systemctl enable nombre-servicio  # Habilitar al inicio
sudo systemctl disable nombre-servicio # Deshabilitar

# Ver logs
journalctl -u nombre-servicio
journalctl -f          # Seguir logs en tiempo real

Monitoreo y mantenimiento:

uname -a               # Información del sistema
free -h                # Memoria disponible
df -h                  # Espacio en disco
uptime                 # Tiempo activo del sistema
last                   # Últimos inicios de sesión

# Logs
cat /var/log/syslog    # Log del sistema
tail -f /var/log/apache2/error.log  # Ver logs en tiempo real

Cron (tareas programadas):

crontab -e             # Editar tareas programadas
crontab -l             # Listar tareas
crontab -r             # Eliminar todas las tareas

# Formato: minuto hora día mes día-semana comando
0 2 * * * /ruta/script.sh          # Cada día a las 2:00 AM
*/30 * * * * comando               # Cada 30 minutos
0 0 * * 0 comando                  # Cada domingo a medianoche
0 9 1 * * comando                  # Primer día del mes a las 9:00

💡 Comandos útiles combinados

# Búsqueda y procesamiento con pipes
ps aux | grep java | awk '{print $2}' | xargs kill    # Matar todos los procesos java

# Buscar archivos grandes
find / -type f -size +100M 2>/dev/null

# Backup automático
tar -czf backup-$(date +%Y%m%d).tar.gz /home/usuario/documentos/

# Contar líneas de código
find . -name "*.py" -o -name "*.js" | xargs wc -l

# Cambiar permisos recursivamente
chmod -R 755 directorio/

# Monitoreo de tráfico de red
netstat -tulpn | grep LISTEN
