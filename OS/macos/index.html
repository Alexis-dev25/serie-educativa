¿Qué es macOS?

macOS es el sistema operativo de Apple para computadoras Mac. Basado en Unix, combina estabilidad con un diseño elegante y es favorito entre creativos y desarrolladores.

Versiones recientes:

  • macOS Sonoma (14): Última versión con IA mejorada
  • macOS Ventura (13): Enfocada en privacidad y rendimiento
  • macOS Monterey (12): Aún soportada, versión estable

¿Por qué es popular para desarrollo?

  • ✅ Basado en Unix (poderosa terminal)
  • ✅ Hardware optimizado (mejor rendimiento)
  • ✅ Excelente para desarrollo iOS/macOS
  • ✅ Herramientas de desarrollo incluidas (Xcode)
  • ✅ Gran comunidad de desarrolladores

📱 Terminal & Zsh

macOS usa Zsh (Z Shell) como shell por defecto desde Catalina.

Comandos básicos (Unix):

ls                     # Listar archivos
ls -la                 # Lista detallada con ocultos
pwd                    # Directorio actual
cd directorio          # Cambiar directorio
mkdir nombre           # Crear directorio
rm archivo.txt         # Eliminar archivo
rm -r carpeta/         # Eliminar carpeta y contenido
cp archivo.txt copia.txt    # Copiar
mv archivo.txt nuevo_nombre # Mover/renombrar
cat archivo.txt        # Ver contenido
open archivo.txt       # Abrir con app predeterminada
open -a "Visual Studio Code" archivo.txt  # Abrir con app específica

Comandos específicos de Mac:

open .                 # Abrir carpeta actual en Finder
open -a Finder         # Abrir Finder
pbcopy < archivo.txt   # Copiar contenido al portapapeles
ls | pbcopy            # Copiar output al portapapeles
pbpaste                # Pegar del portapapeles
screencapture -x screenshot.png  # Captura de pantalla
say "Hola"             # Síntesis de voz
curl -O url            # Descargar archivo

Configurar Zsh:

# Editar configuración
nano ~/.zshrc
# o
vim ~/.zshrc

# Aliases útiles
alias ll='ls -la'
alias python='python3'
alias clearning='clear && printf "\033c"'

# Función de utilidad
mkcd() {
    mkdir -p "$1" && cd "$1"
}

# Cambiar shell
chsh -s /bin/zsh       # Cambiar a Zsh
chsh -s /bin/bash      # Cambiar a Bash

Oh My Zsh (personalización):

# Instalar Oh My Zsh
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

# Usar temas
# Editar ~/.zshrc y cambiar: ZSH_THEME="robbyrussel"
# Temas populares: agnoster, powerlevel10k, spaceship

# Plugins recomendados
# En ~/.zshrc, añadir a plugins array:
plugins=(git zsh-autosuggestions zsh-syntax-highlighting)

🍺 Homebrew - Gestor de Paquetes

Homebrew es el gestor de paquetes estándar para macOS.

Instalación:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Verificar instalación
brew --version
brew doctor             # Verificar salud de Homebrew

Comandos básicos:

brew install nombre-paquete   # Instalar paquete
brew uninstall nombre-paquete  # Desinstalar
brew upgrade           # Actualizar todos los paquetes
brew update            # Actualizar lista de paquetes
brew search búsqueda   # Buscar paquete
brew list              # Listar paquetes instalados
brew info nombre-paquete   # Información del paquete

Paquetes útiles:

brew install node      # Node.js y NPM
brew install python3   # Python 3
brew install git       # Git
brew install wget      # Descargador
brew install nvm       # Node Version Manager
brew install rbenv     # Ruby Version Manager
brew install java      # Java
brew install docker    # Docker
brew install postgresql # PostgreSQL
brew install redis     # Redis

Casks (aplicaciones GUI):

brew install --cask visual-studio-code  # VS Code
brew install --cask google-chrome       # Chrome
brew install --cask firefox             # Firefox
brew install --cask slack               # Slack
brew install --cask discord             # Discord
brew install --cask spotify             # Spotify
brew install --cask notion              # Notion
brew install --cask figma               # Figma

Gestión de servicios:

brew services start postgresql   # Iniciar servicio
brew services stop postgresql    # Detener servicio
brew services restart postgresql # Reiniciar
brew services list               # Listar servicios

💻 Desarrollo en Mac

Xcode Command Line Tools:

xcode-select --install      # Instalar herramientas de línea de comandos
xcode-select --version      # Verificar versión
xcode-select --reset        # Reiniciar configuración

Gestores de versiones:

# Node Version Manager (nvm)
brew install nvm
nvm install node            # Instalar Node.js
nvm use 18                  # Usar versión específica
nvm list                    # Ver versiones instaladas

# Ruby Version Manager (rbenv)
brew install rbenv
rbenv install 3.2.0
rbenv global 3.2.0

# Python con pyenv
brew install pyenv
pyenv install 3.11.0
pyenv global 3.11.0

Desarrollo iOS/macOS:

# Instalar Xcode (desde App Store)
# o línea de comandos
xcode-select --install

# Compilar project Swift
swift build

# Crear proyecto iOS
xcode-select -p         # Verificar path de Xcode
open -a Xcode           # Abrir Xcode

# CocoaPods (gestor de dependencias iOS)
sudo gem install cocoapods
pod init
pod install

Variables de entorno:

# En ~/.zshrc o ~/.bash_profile
export PATH="/usr/local/bin:$PATH"
export PATH="$HOME/.rbenv/bin:$PATH"
export PATH="$HOME/.nvm/versions/node/v18.0.0/bin:$PATH"

# Crear variable personalizada
export MI_VARIABLE="valor"

# Aplicar cambios
source ~/.zshrc

Git en Mac:

brew install git        # Instalar Git (o con Xcode Tools)
git --version
git config --global user.name "Tu Nombre"
git config --global user.email "tu@email.com"

# SSH para GitHub
ssh-keygen -t ed25519 -C "tu@email.com"
cat ~/.ssh/id_ed25519.pub   # Copiar al GitHub

⚙️ Configuración del Sistema

Información del sistema:

system_profiler SPSoftwareDataType   # Info de macOS
sysctl -a               # Parámetros del kernel
uname -a                # Información del sistema
sw_vers                 # Versión de macOS
diskutil list           # Listar discos
df -h                   # Espacio disponible

Procesos y memoria:

ps aux                  # Listar procesos
ps aux | grep nombre    # Buscar proceso específico
top -l 1                # Snapshot de top
Activity Monitor         # Abrir app nativa (GUI)
kill -9 PID             # Matar proceso
killall nombre-app      # Matar aplicación por nombre

Preferencias de sistema:

# Mostrar archivos ocultos en Finder
defaults write com.apple.finder AppleShowAllFiles YES
killall Finder          # Reiniciar Finder

# Acelerar animaciones
defaults write com.apple.dock autohide-time-modifier -int 0
killall Dock

# Tomar screenshots en PNG (cambiar de formato)
defaults write com.apple.screencapture type png

# Listar todos los settings
defaults read

Seguridad:

# Habilitar firewall
sudo /usr/libexec/ApplicationFirewall/socketfilterfw --setglobalstate on

# Exigir password después de dormir
defaults write com.apple.screensaver askForPassword -int 1

# Verificar integridad del sistema
sudo softwareupdate -i -a   # Instalar actualizaciones
spctl --status              # Ver estado de Gatekeeper

Red y conectividad:

ifconfig                # Configuración de red
networksetup -getinfo "Wi-Fi"  # Info de red específica
ping google.com          # Probar conectividad
traceroute google.com    # Ruta de conexión
nslookup google.com      # Resolver DNS
netstat -an             # Conexiones de red
lsof -i :puerto         # Procesos usando puerto específico

Mantenimiento:

# Limpiar caché
rm -rf ~/Library/Caches/*
rm -rf ~/Library/Saved\ Application\ State/*

# Actualizar paquetes Homebrew
brew update && brew upgrade

# Limpiar descargas de Xcode
rm -rf ~/Library/Developer/Xcode/DerivedData/*

# Verificar disco
diskutil verifyVolume /
diskutil repairVolume /
