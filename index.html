Introducción

Bienvenido a mi rincón de código

Esta página nació de una mezcla entre curiosidad, pasión y muchas horas frente a la pantalla. No fue hecha por una empresa ni por un equipo de expertos, sino por mí —un estudiante de 14 años con ganas de entender el mundo de la programación y compartirlo sin filtros ni complicaciones.

Aquí encontrarás los lenguajes más populares, explicados de forma clara, junto con visores de código para que puedas experimentar sin tener que abrir mil pestañas. No hay anuncios, no hay distracciones, solo contenido directo y útil.

¿Está perfecta? No. ¿Está hecha con ganas? Absolutamente.

Porque a veces, lo que empieza como un proyecto escolar termina siendo algo mucho más personal.

Gracias por visitar. Espero que aprendas algo nuevo… o al menos te inspires a crear tu propia versión.

  • JavaScript

    JavaScript es una de las bases de la web moderna otorga interactividad a las páginas web, registra, arroja y guarda información de uso rápido, inputs, botones y resultados.

  • Python

    Python es un lenguaje con la sintaxis más fácil de aprender, es muy parecida al inglés y cuenta con múltiples librerias.

  • SQL

    SQL permite crear una base de datos, su sintaxis se basa en tablas, sería como un excel, pero SQL no cuenta con una interfaz gáfica.

  • C# / CSharp

    C# es un lenguaje muy usado para el desarrollo de apss y programas, funcionando incluso para el desarrollo de apss móviles.

  • VSC

    Visual Studio Code es un editor de código fuente gratuito y potente, que soporta múltiples lenguajes de programación y ofrece características como depuración, contrul de versiones Git integrado y extensiones personalizables.

  • TRAE

    Trae.IA es un entorno de desarrollo integrado (IDE) construido sobre Visual Studio Code y diseñado para ser altamente asistido por IA.

  • Cursor

    Cursor es un editor de código moderno que integra inteligencia artificial para ayudarte a programar más eficientemente.

  • Linux

    Linux es un núcleo de sistema operativo creado en 1991 por Linus Torvalds; sirve como base de distribuciones usadas en servidores y escritorios.
